Lexique phonétique

Générez une URL QR Base45 à partir de mots français, puis décodez-la ensuite en mots.

Cette page recherche chaque mot dans la première colonne de ../lexique_data/lexique_phonetique.csv, compresse les numéros de ligne correspondants dans un jeton Base45 compact, puis génère un QR code pour le lien final. Si la page est ouverte avec ce jeton déjà présent, elle décode les nombres et retrouve automatiquement les mots.
